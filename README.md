# Walmart Sales Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on Walmart sales data using Python. The analysis focuses on identifying sales trends, revenue patterns, customer purchasing behavior, and key business insights from the dataset.

## Features

* Data cleaning and preprocessing
* Missing value and duplicate record handling
* Exploratory Data Analysis (EDA)
* Sales trend analysis
* Revenue pattern identification
* Customer behavior analysis
* Business insights generation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Learning Outcomes

* Data cleaning and preprocessing techniques
* Exploratory data analysis using Python
* Data visualization and interpretation
* Business-oriented analytical thinking
